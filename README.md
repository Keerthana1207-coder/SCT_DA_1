# 📊 SkillCraft Technology – Task 1
## Excel Sales Dashboard

### 📌 Overview
This project was completed as part of the **SkillCraft Technology Data Analytics Internship**. It presents an interactive Excel Sales Dashboard to analyze sales performance and generate business insights.

### 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

### 📊 Dashboard Features
- Total Sales & Profit Analysis
- Category-wise Sales
- Region-wise Performance
- Monthly Sales Trends
- Interactive Filters (Slicers)

### 📈 Key Insights
- Technology category achieved the highest sales.
- West region showed strong sales performance.
- Interactive dashboard enables quick business analysis.
